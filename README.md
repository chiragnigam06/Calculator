# 🧮 Modern Calculator

A modern calculator built with **HTML**, **CSS**, and **JavaScript**, featuring a sleek glassmorphism interface, realistic 3D buttons, smooth animations, and a responsive design for an enhanced user experience.

## ✨ Features

* Basic Arithmetic Operations (+, −, ×, ÷)
* Modern Glassmorphism UI
* Animated Gradient Background
* Realistic 3D Buttons
* Smooth Hover & Click Animations
* Display Glow Effect
* Responsive Design
* Error Handling

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## 🔮 Future Enhancements

* Scientific Calculator Mode
* Calculation History
* Theme Switcher
* Keyboard Support
* Sound & Ripple Effects
